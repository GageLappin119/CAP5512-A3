# CAP5512-A3

To run the code, simply execute:

```
python3 Traveling_Sales_man.py
```
for the code representing the non-fixed start point, run

```
python3 Traveling_Sales_man_fixed.py
```
for the code representing the fixed start point

The file `tsp.dat`, which contains the list of cities and their coordinates, needs to be in the same directory.
